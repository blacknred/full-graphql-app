# Posting app client

## Stack

- NextJS TS bootstrap (create-next-app)
- Urql: Graphql client + graphcache + next-urql(ssr)
- Chakra-ui
- Formik

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
